# Sprint 7 - Perfil e Configurações

## Objetivo

Criar uma área de perfil onde o usuário possa visualizar e editar suas informações básicas.

## Requisitos

- Exibir nome, e-mail, nível atual e score geral.
- Exibir ações concluídas, metas ativas e entradas no diário.
- Permitir editar nome ou apelido.
- Salvar informações por usuário.
- Manter fallback localStorage.
- Não quebrar autenticação, Hoje, Metas, Diário, Dashboard ou Evolução.

## Critérios de aceite

- Perfil carrega dados do usuário autenticado.
- Dados são isolados por usuário.
- Alteração de nome/apelido é salva.
- Lint passa.
- Build passa.
